# react-app

react app

## Framework packages

Vite + React + Redux + React Router + Unocss + Typescript + TDesign React

## Package Manager

pnpm

## Quick start

```
git clone https://github.com/xinlei3166/vite-react-template.git
```

## Project setup

```
pnpm install
```

### Compiles and hot-reloads for development

```
pnpm run dev
```

### Compiles and minifies for production

```
pnpm run build
```

### Run your tests

```
pnpm run test
```

### Lints and fixes files

```
pnpm run lint
```
